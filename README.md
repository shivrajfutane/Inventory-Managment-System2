# Inventory Management System

A modern web based Inventory Management System built to simplify inventory tracking for small and medium businesses. The application helps manage products, categories, suppliers, stock movements, and user authentication using Supabase.

## Features

• Secure user authentication
• Dashboard with inventory statistics
• Product and category management
• Stock In and Stock Out transactions
• Supplier management
• Inventory reports
• Responsive interface
• Real time database with Supabase

## Technology Stack

Frontend:
• HTML5
• Tailwind CSS
• JavaScript

Backend:
• Supabase Authentication
• Supabase PostgreSQL
• Supabase Storage

Libraries:
• Chart.js
